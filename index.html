About GRACE

GRACE (General-purpose Responsive Assistant for Computer Environment) is an AI-powered voice assistant that allows you to control your computer using voice commands. This prototype demonstrates how voice can replace traditional input methods like keyboard and mouse for human-computer interaction.

Developed as a proof-of-concept during a hackathon, GRACE showcases the potential of hands-free computing with cutting-edge AI technology.

Key Features

Voice Command System

Activate with the wake word "listen" and control your computer with natural language commands.

System Control

Open applications, control windows, perform clicks, and more with simple voice commands.

Web Navigation

Open websites and navigate the web using voice commands.

Text Input & Keyboard Control

Type text and execute keyboard shortcuts with your voice.

Mouse Control

Control cursor movement, clicks, and scrolling using voice commands.

Screen Reading

Read text on the screen with OCR technology.

Code Analysis

Analyze and debug code with AI assistance.

AI Generation

Generate images and content using AI with voice commands.

Control with Hand Gestures

Besides voice commands, GRACE can also be controlled using hand gestures captured by your webcam. Perform intuitive hand movements to control your computer without touching the keyboard or mouse.

The hand gesture recognition is powered by MediaPipe and allows for natural interaction with your system.

Cursor Movement

Index finger up - Move your cursor around the screen by moving your index finger

Click Actions

Thumb + Index - Left click

Thumb + Middle - Right click

Scrolling

Index + Middle fingers up - Scroll mode, move hand up/down to scroll

Volume Control

Thumb + Index + Middle - Move hand up/down to adjust volume

Brightness

Index + Middle + Ring - Move hand up/down to adjust screen brightness

Hand Tracking

Advanced tracking with MediaPipe detects 21 points on your hand for precise control

Technologies Used

Core Technologies

  • Python - Main programming language
  • PyAudio & SpeechRecognition - For speech processing
  • PyAutoGUI - For system control
  • Tesseract OCR - For screen reading
  • Google Gemini API - For AI capabilities
  • MediaPipe - For hand gesture recognition

System Integration

  • PyWin32 - Windows-specific functionality
  • PyStray - System tray integration
  • NSSM - Service management
  • Windows Shell Protocol - For UWP apps

System Requirements

  • Windows 10/11, macOS, or Linux
  • Python 3.7+
  • Microphone
  • Internet connection
  • Tesseract OCR installed

Installation Steps

  1. Download the zip file
  2. Extract the contents
  3. Run pip install -r requirements.txt
  4. Execute python main.py to start
